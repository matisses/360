Administrar Proveedor

Para administrar los datos, consignaciones y proformas de un proveedor; escriba el código o nombre de este en el campo de filtro, para buscarlo. Luego de colocar el parámetro de búsqueda de Enter para ejecutar la consulta.

Una vez ejecutada la consulta se desplegara un dialogo con los resultados relacionados con la búsqueda; el dialogo se verá como la siguiente imagen.

Luego de tener los datos y de encontrar el proveedor que se necesita, de clic sobre este y se cargaran los datos correspondientes al proveedor.

Cada celda que contiene el proveedor está compuesta por 2 características:

1 - Imagen Se le mostrara al usuario el logo que se encontró del proveedor; si este no posee logo se mostrara una imagen estándar.

2 - Código proveedor El código que le corresponde a cada proveedor.

Al usuario seleccionar un proveedor, se cargaran en la parte inferior unos tags:

Datos proveedor

Contenedores

Transacciones bancarias

Anticipo cuenta

A continuación, se explicara uno a uno de los tags y que acciones podemos realizar.

Datos Proveedor

En este tag se podrán editar los datos del proveedor, vea Campos.

Una vez se haya modificado alguno de los anteriores datos, el usuario debe dar clic en Actualizar y los cambios quedaran guardados; si el usuario no quiere realizar ningún cambio basta con no dar clic.

El usuario puede también cambiar el logo del proveedor; pero es de tener en cuenta que una vez seleccionada una imagen para cambiar, esta quedara cambiada automáticamente. Para ver más acerca de imágenes vaya a Imágenes.


Imágenes

Para asignar una imagen al proveedor siga los siguientes pasos:

1. De clic en Navegar, como se muestra en la siguiente imagen.

2. Se abrirá una ventana emergente del explorador de archivos de Windows.

3. Seleccione una imagen con una de las siguientes extensiones: png, jpg o jpeg y de medidas como máximo de 260px * 260px.

4. La foto será asignada automáticamente.

Contenedores

El tag de contenedores, permite gestionar los contenedores de cada proforma permitiendo modificar los datos que se explicaran a continuación:

Campo Descripción Modificable
#{datoContenedor[0]} #{datoContenedor[1]}

Además, se permite al usuario registrar un comentario para la proforma desde este tag. También se muestra un resumen de cuantos contenedores se han asignado a la proforma, tal y como se muestra en la siguiente imagen.

Nota: Es muy importante tener en cuenta que por este medio no se pueden agregar más contenedores a la proforma, para agregar más contenedores debe hacerlo por medio del módulo de proformas.

Transacciones Bancarias

Este tag permite la administración y gestión de los pagos realizados a una proforma; aquí el usuario puede crear, usar saldos, asignar fechas de giros, visualizar una transacción, ver pdf y re-generar pdf.

A continuación, se explicaran cada una de las funcionalidades mencionadas anteriormente:

  • Crear transacción Para crear una nueva transacción a una proforma se debe tener en cuenta las siguientes exigencias:

    Saldo pendiente La proforma debe tener saldo pendiente por pagar, para poder realizar este.

    Moneda Se debe asignar una moneda a usar para la proforma; esto ayuda a que las transacciones queden registradas con esta.
    Nota: Por este módulo no se puede asignar monedas a las proformas, para asignar estas debe realizarlo por el módulo de proformas.

    Cuentas El proveedor debe tener al menos una cuenta asignada al proveedor. Para asignar cuentas a un proveedor vaya a Crear cuenta.

    Luego de tener los datos necesarios procedamos con la creación de una nueva transacción. Dando clic en el botón

    Al dar clic en el botón anterior, abrirá un dialogo como el de la siguiente imagen y con las siguientes características:

    1 - Cuenta bancaria El usuario debe seleccionar una de las cuentas bancarias disponibles para el proveedor.

    2 - Fecha giro Si se dispone de la fecha en que se va a girar el dinero ingrésela, para esto seleccione una fecha del calendario que se despliega al dar clic en el campo de fecha.

    3 - Tipo giro Seleccione un giro de la lista de giros, a continuación se explica cada tipo:

    Anticipo: Es cuando se le va a girar al proveedor un porcentaje del dinero total.

    Balance: Se usa cuando se va a pagar el total que se debe de la proforma.

    Roll on: El usuario puede usar este tipo siempre y cuando, se disponga de anticipo en cuenta a usar de la cuenta seleccionada y de la moneda de la proforma.

    4 - Anticipo $ Si se selecciona el tipo de giro Anticipo, se habilitara este campo para permitir la asignación de un valor a girar de la moneda de la proforma; recuerde que aquí se debe insertar un valor inferior al total de la proforma.

    5 - Anticipo % Una vez se haya ingresado un Anticipo $, se calculara automáticamente el porcentaje de anticipo a girar.

    6 - Balance $ Al seleccionar el tipo de giro Balance, se habilitara el campo que contendrá el total a girar.

    7 - Total $ Aquí se verá el total a girar, al seleccionar el tipo de giro Balance, se permitirá insertar un valor superior a lo que se debe; la diferencia será registrada como Anticipo cuenta.

    8 - Comentario Ingrese un comentario para justificar la transacción.

    Una vez todos los datos estén respectivamente diligenciados, el usuario debe dar clic en Guardar para que la transacción se genere, o de clic en Cancelar si no quiere que los datos queden almacenados. Es de recordar que si el usuario no ingresa la fecha, la transacción quedara en un estado pendiente hasta que esta sea asignada.

    Luego de que se guarde la transacción, se generara el documento en PDF y le permitirá al usuario la visualización de este y para que lo pueda imprimir, si es necesario.

  • Usar saldos Para usar saldos de anticipos en cuenta mire el proceso de Crear transacción en específico el tipo de giro Roll on.

  • Visualizar transacción Si se quiere visualizar una transacción realizada, de clic en el botón . Al dar clic se abrirá el mismo dialogo para crear, pero en este únicamente se podrá modificar la fecha de giro de la transacción, esto lo veremos en la siguiente imagen.

  • Ver PDF El usuario puede ver el documento que genera el sistema para la transacción, al dar clic en el anterior botón se abrirá en otra ventana de su explorador el pdf, el cual contiene datos importantes sobre la transacción; como se podrá apreciar en las siguientes imágenes:

    En la anterior imagen se puede apreciar el encabezado del documento que se compone de la información principal que identificara cada transacción.

    1 - Logo de la marca.

    2 - Información de la compañía.

    3 - Número de la transacción.

    4 - Fecha y número de la proforma.

    En esta otra imagen se pueden ver los datos del proveedor.

    1 - Datos principales del proveedor.

    2 - Logo del proveedor Si el proveedor no posee logo no se verá nada.

    3 - Espacios para firmas de aprobación.

    La imagen anterior es la que contiene los datos del banco y la cuenta a la que se va a realizar el giro.

    1 - Datos del banco y la cuenta.

    2 - Espacios para firmas de aprobación.

    En esta imagen el usuario encontrara los datos de la transacción para el pago.

    1 - Valores a girar dependiendo del tipo de giro.

    2 - Descripción para la transacción.

    3 - Espacios para firmas de aprobación.

    4 - Tipo de giro y valor total a girar.

    La imagen anterior es el pie del documento y es donde se encontraran las firmas de autorización de todos los implicados en el proceso de pago.

  • Re-generar PDF Si el sistema no fue capaz de generar el archivo PDF, el usuario puede dar clic en el siguiente botón , para generar de nuevo el documento que corresponda a la transacción creada anteriormente.

Recuerde que por proforma el sistema mostrara cual es el saldo Total, Pendiente, Transferido y Moneda, que se están usando para esta. Lo anterior con el fin de facilitar al usuario los datos necesarios para realizar el control de pagos al proveedor por proforma.

Anticipo Cuenta

El siguiente tag funciona de forma que al usuario le sea más cómodo controlar el flujo de los saldos en cuenta que se le han consignado al proveedor o usado en proformas. En este paso vamos a explicar como agregar y cómo usar este saldo:

  • Agregar Para poder agregar un saldo a cuenta para el proveedor, se debe dar clic en el siguiente botón

    Una vez el usuario haya dado clic, se abrirá una ventana emergente en la que se podrá llevar a cabo el registro de la transacción por motivo de Anticipo a cuenta. Ahora se explicara de que se compone y como se crea un anticipo:

    1 - Valor anticipo cuenta: Aquí el usuario debe ingresar el valor que desea se agregue a anticipo en cuenta.

    2 - Moneda: Seleccione de la lista de monedas, que moneda es la que va a usar para la transacción.

    3 - Cuenta bancaria: Seleccione de la lista, la cuenta que se va a usar para efectuar la transacción.

    4 - Comentario: Ingrese un comentario que ayude a identificar porque motivo se realizó la transacción.

    Una vez tenga los datos respectivamente diligenciados de clic en el botón Crear para realizar la transacción, de lo contrario de clic en Cancelar para cerrar el proceso. Después de haber creado la transacción se verán los movimientos en una tabla con los siguientes datos:

    Campo Descripción
    #{datoTablaAnticipo[0]} #{datoTablaAnticipo[1]}

    Además de los anteriores campos definidos, también se verán los siguientes iconos , que significa que el movimiento fue una entrada de anticipo a cuenta; , es cuando el tipo de movimiento fue un uso de anticipo a una proforma.

    Cuando se crea una entrada de anticipo a cuenta, esta quedara como pendiente hasta que el usuario le asigne una fecha de giro; una vez puesta esta fecha la transacción quedara como habilitada. Para asignar la fecha de clic en el botón , el cual abrirá nuevamente la ventana emergente, con los campos bloqueados para evitar la modificación de este; pero únicamente estará habilitada el campo de asignación de fecha, una vez colocada de clic en el botón Actualizar lo que implicara que la transacción quede activa. Después de que el usuario ingrese la fecha, el botón para visualizar quedara deshabilitado para la transacción. Si el usuario no quiere asignarle la fecha a la transacción de clic en Cancelar.

  • Quitar Para poder quitar un saldo a cuenta para el proveedor, se debe dar clic en el siguiente botón

    Una vez el usuario haya dado clic, se abrirá una ventana emergente en la que se podrá llevar a cabo el descuento de la transacción por motivo de Anticipo a cuenta. Ahora se explicara de que se compone y como se quita un anticipo:

    1 - Valor anticipo cuenta: Aquí el usuario debe ingresar el valor que desea se quite de anticipo en cuenta.

    2 - Proforma: Seleccione de la lista de proformas, la proforma que se va a usar para la transacción.

    3 - Cuenta bancaria: Seleccione de la lista, la cuenta a la que se le va a quitar el valor.

    4 - Comentario: Ingrese un comentario que ayude a identificar porque motivo se realizó la transacción.

    Después de que haya llenado los datos solicitados, de clic en el botón Crear para realizar la transacción; de lo contrario de clic en el botón Cancelar para cerrar el proceso. Se debe tener en cuenta que a diferencia del agregado de anticipo a cuenta, este ingresara la fecha de transacción automáticamente y, se creara el registro de pago en el proceso de Transacciones bancarias.

Crear Cuenta Bancaria

Para crear una cuenta nueva o asignar una cuenta al proveedor actual de clic en el botón como el de la siguiente imagen.

Al dar clic en el anterior botón, se abrirá una ventana emergente compuesta de 3 pasos para poder crear o asignar una cuenta bancaria.

  • Paso 1 - Banco de la cuenta.

    1 - Razón social banco: Nombre legal del banco.

    2 - Bancos: Lista de bancos disponibles o cree un nuevo banco.

    El usuario debe seleccionar uno de los bancos disponibles en la lista de bancos, para poder gestionarlo, selecciónelo para que la razón social se llene y pueda editar el nombre del banco, después de que lo cambie de clic en Guardar; o seleccione Nuevo banco e inserte el nombre del banco y de clic en guardar; una vez seleccione un banco se activara el botón de Siguiente, de clic en este para ir al próximo paso.

  • Paso 2 - Sucursal bancaria.

    1 - Dirección sucursal: Lugar donde se encuentra ubicada la sucursal del banco.

    2 - Swift: Confirmación de giro.

    3 - País: País de la sucursal del banco.

    4 - Estado: Estado del país de la sucursal.

    5 - Ciudad: Ciudad del estado de la sucursal.

    6 - Sucursales: Lista de sucursales asociadas al banco o cree una nueva sucursal.

    Seleccione una sucursal existente para que se carguen todos los datos relacionados a la sucursal y si es la necesidad, cámbielos y de clic en Guardar para que se almacenen los cambios; o seleccione Nueva sucursal y llene todos los datos necesarios para crear la sucursal del banco, una vez llene los datos no olvide dar clic en Guardar; una vez seleccione una sucursal se activara nuevamente el botón de Siguiente, de clic en este para ir al próximo paso.

  • Paso 3 - Cuenta bancaria.

    1 - Cuenta bancaria: Número de la cuenta bancaria.

    2 - Iban: Serie de caracteres alfanuméricos que identifican una cuenta determinada en una entidad financiera.

    3 - Aba: Serie que identifican una cuenta determinada en una entidad financiera.

    4 - Cuentas: Lista de cuentas asociadas a la sucursal o cree una nueva cuenta.

    Al seleccionar una cuenta existente se cargaran los datos correspondientes a esta y cambie los datos si es necesario, modifique y de clic en Guardar para almacenar cambios o seleccione Nueva cuenta y llene los datos que se necesitan y de clic en Guardar. Una vez realizado esto la cuenta quedara relacionada al proveedor.

En los pasos 2 y 3 el usuario puede devolverse a los pasos anteriores dando clic en el botón Anterior y de clic en el botón Cerrar, para finalizar el proceso y usar la nueva cuenta.

Exportar Informe

Para exportar el informe sobre las proformas en producción, aquí se muestra el tipo de contenedor, fechas de embarque y pagos. Para realizar la exportación de clic en el botón con el siguiente icono , una vez de clic se abrirá una ventana emergente en la que se pedirán los parámetros de exportación; estos parámetros los veremos a continuación:

El primero de todos los parámetros es el tipo de exportación en el que se verán dos opciones:

1 - General: Es cuando el usuario quiere que el informe se descargue de todos los proveedores y de todas las proformas.

2 - Proveedor: Se usa cuando se quiere ver el informe del proveedor en el que se está trabajando en específico; para poder seleccionar esta opción debe ingresar un proveedor como se explica en Administrar Proveedor.

Si el usuario selecciona general, se habilitara otro parámetro.

En este caso se tendrán tres opciones a seleccionar:

1 - Seleccione: Si el usuario quiere que se muestren todos los datos, este debe dejar el filtro en el valor por defecto.

2 - Fecha: Cuando se quiere ver únicamente los registros que hay para la fecha seleccionada, esta opción habilitara un campo con un calendario para seleccionar la fecha.

3 - Vencidas: Se mostraran todas las proformas que se encuentran vencidas en General para todos los proveedores, o en Proveedor para las de ese en específico.

Al seleccionar en el primer filtro, la opción de Proveedor; se habilitara el filtro de Tipo dato como se muestra en la anterior imagen. Esta imagen al igual que las demás nos muestra opciones para exportar, en este caso 2:

1 - General: El informe se generara con todas las proformas del proveedor.

2 - Proforma: Se usa cuando el usuario quiere el informe de una proforma específica.

Una vez seleccionado el tipo de datos Proforma, se habilitara nuevamente otro filtro; en este caso el de las proformas donde se listaran todas las disponibles para el proveedor.

Después de que el usuario haya seleccionado los filtros correspondientes, se habilitara el botón de Exportar, de clic en este y comenzara la descarga del archivo en un libro de excel, en cualquier momento puede dar clic en Cancelar.
