Registrar Compra Lista de Regalo

En este módulo le permite al asesor interactuar con las lista de regalos de la mano del cliente, para seleccionar que productos desea adquirir de estas. A continuación se expone el modo de uso.

Contenido

  • Consulta de Datos
  • Seleccionar Productos
  • Mensajes del Sistema



Consulta de Datos

Es necesario saber el número de la factura para luego pasar a consultar las listas de regalos; solo se muestran las listas de regalos activas y que no están vencidas.
Ver imagen 1: Primer parámetro de consulta.

[Imagen 1]

Una vez digitada la factura de venta, se debe oprimir la tecla ↔ tabular para que el sistema despliegue el segundo parámetro de búsqueda, que se utilizara para ingresar ya sea el nombre de la lista, el código, el creador o cocreador de la lista de regalo.
Ver imagen 2: Segundo parámetro de consulta.

[Imagen 2]

Si el sistema encuentra lista de regalos que coincidan con el segundo parámetro de consulta ingresado por el usuario, este presta la funcionalidad de poder escoger que lista es realmente se va a usar.
Ver imagen 3: Listas de Regalos Encontradas.

[Imagen 3]

Seleccionar Productos

Una vez se consulten los datos, se despliega la acción de seleccionar productos. Aquí se visualizan los productos que el cliente solicitara seleccionar para registrarlo a la lista de regalo.
Ver imagen 4: Seleccionar Productos.
Recomendación: Antes de hacer clic en Guardar verifique que la factura y la lista de regalo correspondan a la que el cliente desea registrar.

[Imagen 4]

En la imagen 4, el punto 1 contiene información básica del producto, adicional se informa la cantidad faltante por regalar en la lista de regalo. El punto 2 se ofrece dos acciones como funcionalidad adicional, la primera es para agregar el producto cuando el botón este en negativo y la segunda es para eliminar cuando botón este en positivo; para cualquiera de estas dos acciones debe seleccionar el producto y hacer clic en el botón. En el punto 3 están disponibles los botones de Cancelar el registro o Guardarlo.
Al hacer clic en el botón Guardar el cliente puede dejarle un comentario al dueño de la lista de regalo, si él lo desea, de lo contario hacer clic en el botón Terminar se finalizara el proceso de registro.
Ver imagen 5. Terminar Registro.

[Imagen 5]
