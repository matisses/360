Cargar Proforma

En esta página se le permite al usuario realizar la respectiva carga de la proforma después de ser recibida por el proveedor; además de permitir la asignación de nuevas columnas o de columnas existentes a las creadas por el proveedor; también se le permitirá al usuario la eliminación de las columnas que se consideren no serán usadas en la proforma. Se debe tener en cuenta que todas las modificaciones se guardaran en una versión diferente a la plantilla pre-definida para el proveedor.

Tenga en cuenta lo siguiente: La configuración del encabezado de la PI no puede cambiar en cuanto a su estructura, ya que esta está diseñada de acuerdo a la plantilla de muestra y el encabezado es leído tal cual como fue configurado. Este paso es súper importante tenerlo en cuenta debido a que de aquí se tienen los datos del proveedor, lo que permitirá una asociación exacta de la PI con el Proveedor.

Después de tener la anterior información clara, procedamos a cargar nuestra proforma.

Seleccionar Proforma

Para crear una nueva proforma en el sistema se debe primero cargar el archivo de excel que tiene los datos de la carga, para esto en los siguientes pasos se explicara como subir el archivo y las condiciones que este debe tener para que el sistema no se confunda con los datos.

1 - En la parte superior de la vista, el usuario puede visualizar un componente como el que se muestra en la Imagen 1.

En la imagen se muestra un componente con un botón Examinar..., de clic en este.

[Imagen 1]

2 - Después de dar clic en el anterior botón, se nos abrirá el explorador de archivos de Windows; seleccione el archivo que desea subir y de clic en Abrir. Luego de haber seleccionado el archivo se deberá ver como lo muestra la Imagen 2.

[Imagen 2]

3 - Una vez seleccionado el archivo de extensiones .xlsx o .xls, el usuario puede dar clic en el botón Cargar para que el sistema comience el escaneo del archivo y por consiguiente se puedan ver los datos.

El usuario puede ver al cargar la proforma que se muestran los datos del proveedor y las columnas que se pudieron detectar del archivo.

Datos Proveedor

En estos datos se verá la información de contacto y la razón social del proveedor, así como el logo de este.

[Imagen 3]

Columnas

El sistema detectara automáticamente las columnas del archivo y se las mostrara al usuario para que este decida si desea incluirlas o ignorarlas.

Se pueden tener 3 tipos de columnas que se explicaran a continuación:

  • Fondo gris Son las columnas que se encontraron en el archivo y existen en las columnas del sistema, el sistema es capaz de detectar esto automáticamente para facilitar el trabajo del usuario; además de que así se verán también las columnas asignadas.

    [Imagen 4]

  • Fondo blanco Estas columnas no fueron asociadas a ninguna existente como en el primer tipo, estas columnas permiten la asignación con columnas del sistema.

    [Imagen 5]

  • Fondo verde Las columnas con este color son más características, porque están indicando que son obligatorias ósea que no importa si tiene o no datos estas serán agregadas automáticamente.

    [Imagen 6]

Después de tener claro cada tipo de columna, ahora empezaremos a configurar la forma como se quiere ver posteriormente la proforma y la asignación de las columnas.

Asignación y Configuración

Lo más importante en este módulo es la facilidad de configuración y asignación de las columnas obtenidas.

Asignación

Para poder asignarle una columna existente a una columna obtenida del archivo cargado, de clic sobre la columna y esta quedara de un color rojo como se muestra en la Imagen 7.

[Imagen 7]

Lo anterior permite trabajar sobre la columna seleccionada, además se habilitara un botón identificado con el siguiente icono , dando clic en este se abrirá una ventana emergente como la de la Imagen 8.

[Imagen 8]

1 - Crear columna Seleccione esta opción para crear una columna. Para saber más sobre este tema, vaya y mire el manual de columnas.

2 - Usar columna Seleccione una columna de la lista para asignarla.

De clic en Guardar para que los cambios queden almacenados o de lo contrario de clic en Cancelar.

Configuración

En este paso el usuario puede dar posición o eliminar una columna que crea innecesaria; para eliminar la columna de clic en el botón con el siguiente icono y esta se eliminara. Use los botones identificados con los iconos y , para mover las columnas; recuerde que las primeras 5 columnas se mostraran como más importantes en la proforma.

Crear Proforma

Una vez se tengan las columnas como se quieren en la proforma de clic en Crear, para que la proforma se genere o de clic en Cancelar para no hacer nada y limpiar la zona de trabajo.
