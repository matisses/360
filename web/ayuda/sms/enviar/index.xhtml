Enviar

Para enviar un mensaje se necesitan 2 datos muy indispensables:

  • Número celular: Digite el número del celular al que quiere enviarle el (los) mensaje(s); además también puede ingresar en este campo el Nit del cliente, ingrese el Nit y de Enter para realizar la búsqueda y se mostraran las líneas disponibles, seleccione la que desee usar. Si desea enviar el mensaje a más de un destinatario separe estos con ; (Punto y coma), sin espacios.
  • Mensaje: Ingrese el texto que quiere enviarle al cliente; recuerde que cada mensaje se compone de 150 caracteres, pero no significa que no pueda enviar mensajes más largos. Cuando este ingresando el mensaje para el cliente inmediatamente en la parte inferior de la caja del mensaje y en la parte superior de los botones, aparecerá una burbuja con el mensaje y cada vez que se supere el límite de 150 se creara una de estas burbujas; esto es con el fin de que el usuario este consiente de cuantos mensajes enviara.
