Movimiento Entre Ubicaciones

Este módulo le permite al usuario realizar movimientos de la mercancía localizada en el almacén en el que el usuario inició sesión. Se permite realizar movimientos de una ubicación a otra sin necesidad de que esta pase por la ubicación de piso (TM).

Condiciones de Uso

Para mover un artículo entre ubicaciones diferentes de piso (TM), se debe tener en cuenta las siguientes observaciones:

  • Primero Los movimientos son únicamente del almacén en el que se inició sesión, no se permite y no sirve para realizarlos de almacén a almacén.

  • Segundo El modulo se debe usar si y solo si, la mercancía no tiene que pasar por piso (TM), de lo contrario se debe usar las funciones de Cargue y Descargue según correspondan.

  • Tercero Esta aplicación es para el uso del personal de tiendas y/o para los que posean el permiso de usarlo.

Crear Movimiento entre Ubicaciones

Después de tener claro todas las condiciones que aplican para este módulo y de tener claro su funcionalidad, procederemos a explicar los pasos y la forma de usar este módulo.

Para mover artículos de una ubicación a otra, se necesitan los siguientes datos:

  • Ubicación Origen Código de la ubicación de la que se quiere sacar la mercancía.

  • Ubicación Destino Código de la ubicación a la que entrara la mercancía.

  • Referencia Código del producto a descargar. Existen dos métodos para ingresar la referencia:

    1 - Referencia completa Es necesario que se ingresen los 20 caracteres de la referencia. Este método se usa normalmente al escanear los códigos de barras de los productos.

    2 - Referencia corta Sólo los usuarios con permiso podrán ingresar referencias cortas. Una referencia corta está compuesta por los tres primeros dígitos (código del proveedor), un separador (asterisco o punto) y los últimos dígitos (consecutivo del producto dentro del proveedor). Ejemplos: 105*1022, 223.10, 228*5, 117.0015. Los usuarios que no tengan autorización para ingresar referencias cortas recibirán un mensaje indicándoles que no tienen permiso para hacerlo, y deberán usar el proceso de referencia completa.

Una vez ingresadas tanto la ubicación origen como la ubicación destino y la referencia, se debe dar clic en el botón Agregar para incluir la referencia al movimiento. Al ingresar la primera referencia los campos de ubicación se bloquearan para evitar que estas se cambien, ya que no se puede mover en un solo traslado, de varias ubicaciones. OJO Lo anterior no implica que al darle agregar se realice el respectivo movimiento.
