Crear

Para crear un traslado de este tipo, el usuario debe seleccionar el almacén desde el que se realizara el traslado y seleccionar el almacén al que se realizara el traslado.

Luego de que haya seleccionado los dos ítems anteriores, se habilitaran las ubicaciones tipo Pallet que estén creadas en el almacén origen y que tengan saldo en el momento.

Para identificar de una forma más adecuada la selección de una de las ubicaciones Pallet, vea la siguiente información:

  • Como en esta imagen el usuario verá las ubicaciones Pallet que no ha seleccionado.

  • En la anterior imagen se puede evidenciar que la ubicación fue seleccionada para trasladar, para seleccionar una ubicación simplemente de clic sobre esta y listo, de igual forma para deseleccionarla vuelva a darle clic.

Una vez seleccione las ubicaciones que desea, de clic en Trasladar para generar el traslado o de clic en Cancelar para no hacer nada.

Si desea que se imprima el traslado que se generara, seleccione la opción Imprimir traslado y se imprimirá el documento en la impresora de la sucursal donde inicio sesión.
