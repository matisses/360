Columna Calculada

Para realizar un trabajo completo con las columnas, estas tienen la posibilidad de crearse a partir de una operación que se auto calcularán, cada que sus datos cambien de valor.

Para crear una columna calculada, de clic en el check que tiene el mismo nombre y se abrirá un dialogo, que es en el que se configurara la operación.

Al usuario se le pedirán 4 datos indispensables para la creación de la operación:

1 - Operando 1 En este dato se exige que se use una columna de las existentes para realizar la operación.

2 - Tipo operando 2 El dato 2 es una lista en la que se dan dos opciones:

1 - Columna Se puede usar una segunda columna para realizar la operación.

2 - Constante Se puede usar un valor estático para realizar la operación; Ejemplo: Valor Total * 15, siendo 15 la constante.

3 - Operación En este dato el usuario debe seleccionar el tipo de operación que llevara a cabo. Ejemplo: -, +, etc.

4 - Operando 2 Seleccione o digite el operando 2, sea columna o constante.

Una vez se tengan los datos con los que se realizara la operación de clic en el botón Guardar. Si el usuario no desea que la operación se guarde, debe dar clic en el botón Cerrar, para cancelar el proceso.

Después de haber guardado la operación, se vera de la siguiente forma, tal y como se muestra en la imagen. Si quiere modificar la operación de clic sobre esta y modifíquela.

Nota: Operando es un elemento sobre el cual se aplica una operación.

Creación

Llene todos los datos definidos anteriormente para poder crear una columna nueva. De clic en Crear.

Modificación

Si el usuario desea cambiar las propiedades o los datos de la columna, seleccione una de estas, modifique los cambios que debe modificar y de clic en Editar.

Eliminación

Si el usuario desea eliminar la columna, seleccione una de estas y de clic en Eliminar.
