Histórico Mensajes de Texto

En este módulo, el usuario podrá ver y filtrar los mensajes que se han mandado por todos los usuarios.

Una vez el usuario ingrese a esta opción, podrá ver todos los mensajes que se han enviado y su contenido; para ver más a cerca del contenido de los mensajes vaya a visualización de mensajes.




Contenido

  • Mensajes Recibidos



Consultar

Si no se quieren ver todos los mensajes, de todos los números de celular; el usuario puede filtrar por:

  • Celular: Digite el número del celular que quiere ver y obtener los datos.
  • Texto: Ingrese un texto por el que quiere filtrar, y obtendrá todas las coincidencias con este.

Para poder tener los resultados de lo que quiere buscar, es muy importante que una vez digite los datos en el campo de Parámetro búsqueda, de clic Enter para ejecutar la consulta.

Visualización de mensajes

El usuario vera cada mensaje en una burbuja como la que se mostrara y detallara en la siguiente imagen:

  • 1 - Fecha y hora en la que se envió el mensaje al cliente.
  • 2 - Número de celular al que se mandó el mensaje.
  • 3 - Foto del usuario que envió el mensaje.
  • 4 - Usuario que mando el mensaje.
  • 5 - Estado del mensaje, en este paso se podrán ver 3 iconos que identificaran que sucede con el mensaje.
    • El mensaje fue enviado exitosamente.
    • No fue posible enviar el mensaje.
    • Ocurrió un problema no identificado con el envió del mensaje.
  • 6 - Mensaje que el usuario escribió para que le llegue al destinatario.
  • 7 - Burbuja contenedora. Nota: A cada usuario se le asignara un color con el que se identificaran los mensajes que este envió.
