360 - Ayuda Búsqueda Prestashop

Consulta productos SAP-Prestashop

En esta página se le permite al usuario realizar consultas que validen si un producto está siendo enviado a la página web. La consulta se puede realizar de dos maneras, que se explican a continuación.

Búsqueda por referencia

El usuario puede ingresar una referencia completa o abreviada para realizar la consulta. Sólo se mostrará como resultado el producto consultado con todos sus atributos y respectivos valores, si es que dicho producto se encuentra habilitado para web. Los siguientes son ejemplos de búsquedas por referencia.
Entrada Resultado
107*123 La referencia se completa usando el asterisco como comodín.
105.1243 La referencia se completa usando el punto como comodín.
11800000000000000908 Referencia completa, se busca sin modificaciones.
118-209 Se busca como modelo, ya que el caracter guión no se considera un comodín válido.
1050000000000000192 Se usa como modelo, ya que el texto tiene menos de 20 caracteres y no contiene ningún comodín válido.

Búsqueda por modelo

Cuando el texto ingresado no puede ser identificado como una referencia, se asume que se ingresó un modelo. Cuando se ejecuta la búsqueda por modelo, el sistema primero consulta todas las referencias que pertenecen a dicho modelo y luego, internamente, realiza la búsqueda por referencia.

Condiciones para que un producto se encuentre publicado en la página web

Las siguientes son las condiciones que debe cumplir un artículo para que éste sea visible en la página web de Matisses:
  • El producto debe tener saldo disponible para la venta. Esto significa que debe tener al menos una unidad en algún almacén cuyo código comience con el número cero.
  • El producto debe tener la información básica para web completa. Esta información incluye los campos nombre web, descripción, descripción corta, departamento, grupo, subgrupo, materiales (1), colores, combinación principal, nombre, nombre extranjero, palabras clave, modelo, precio de venta, fecha de procesamiento de imágenes (2) y activador para web.
  • Se debe haber copiado previamente la carpeta con fotos en la ruta indicada del servidor hercules.

(1) Aunque los materiales no son obligatorios, se recomienda que se encuentren habilitados ya que al realizar esta relación se cargan también de manera automática los cuidados del producto.

(2) Se requiere que la fecha de procesamiento de imágenes sea la fecha en la cual se va a realizar la sincronización cuando el producto se va a cargar por primera vez a la página web. Si es un producto que ya se ha cargado con anterioridad, esta fecha solo debe ser actualizada si se desean reprocesar las fotos para el producto.

Resultado de búsqueda

Una vez se realice una búsqueda usando una referencia o un modelo válidos, el sistema mostrará de manera estructural la información completa de los productos encontrados. Los atributos que se muestran son construidos de manera dinámica y por lo tanto pueden diferir con los de la imagen de ejemplo. Asimismo, la estructura mostrada depende del dispositivo que se use, ya que ésta se ajusta automáticamente a la resolución del equipo del usuario.
