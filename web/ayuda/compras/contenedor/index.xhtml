Contenedores Proveedor

En este módulo el usuario puede asignarle o quitarles contenedores a los proveedores, para que estos trabajen únicamente con los seleccionados.

  • Mensajes Recibidos



Seleccionar Proveedor

Para realizar las operaciones de agregado y eliminado, es muy importante primero que todo aprender como seleccionar el proveedor al que se le quiere asignar o quitar el contenedor; para esto haga lo siguiente.

Ingrese el código del proveedor en el contenedor que se encuentra en la vista; este contenedor es igual al que se muestra en la siguiente imagen. Luego de Enter para ejecutar la consulta.

Una vez encontrado el proveedor, se desplegara una lista con los contenedores disponibles para ser agregados y los que ya han sido agregados al proveedor.

Agregar Contenedor

Para agregar un contenedor al proveedor siga los siguientes pasos:

1 - Seleccione un contenedor de los disponibles en la lista.

2 - De clic en el botón Agregar para poder incluirle en contenedor al proveedor.

Luego de que agregue el contenedor, este se verá en un contenedor como el siguiente. En el contenedor se ve el nombre del contenedor que se agregó.

Quitar Contenedor

Para quitar un contenedor al proveedor haga lo siguiente:

1 - Pase con el cursor del mouse por encima del contenedor; una vez hecho esto el contenedor tomara un color rojo y aparecerá un icono como el siguiente , lo anterior se puede observar en la siguiente imagen.

2 - De clic en el botón con el siguiente icono para proceder con la eliminación.

El sistema dejara eliminar únicamente el contenedor, solo si este no está siendo usado por alguna proforma.
