Movimiento de Descargue de Ubicación

Este módulo le permite al usuario realizar movimientos de la mercancía localizada en el almacén en el que el usuario inició sesión. Se permite en este módulo realizar únicamente movimientos de ubicación a piso (TM).

Crear Movimiento de Descargue

Para descargar un producto que se encuentra en una ubicación a piso (TM), se debe suministrar la siguiente información:

  • Ubicación Código de la ubicación de la que se quiere sacar la mercancía.

  • Referencia Código del producto a descargar. Existen dos métodos para ingresar la referencia:

    1 - Referencia completa Es necesario que se ingresen los 20 caracteres de la referencia. Este método se usa normalmente al escanear los códigos de barras de los productos.

    2 - Referencia corta Sólo los usuarios con permiso podrán ingresar referencias cortas. Una referencia corta está compuesta por los tres primeros dígitos (código del proveedor), un separador (asterisco o punto) y los últimos dígitos (consecutivo del producto dentro del proveedor). Ejemplos: 105*1022, 223.10, 228*5, 117.0015. Los usuarios que no tengan autorización para ingresar referencias cortas recibirán un mensaje indicándoles que no tienen permiso para hacerlo, y deberán usar el proceso de referencia completa.

Una vez ingresadas tanto la ubicación como la referencia, se debe dar clic en el botón Agregar para incluir la referencia al movimiento. Al ingresar la primera referencia el campo de ubicación se bloqueara para evitar que esta se cambie, ya que no se puede mover en un solo traslado, de varias ubicaciones. OJO Lo anterior no implica que al darle agregar se realice el respectivo movimiento.

Luego de tener todas las referencias necesarias para el movimiento, dé clic en el botón de Descargar y espere la respuesta del sistema, el cual indicará si fue o no exitoso el movimiento.
