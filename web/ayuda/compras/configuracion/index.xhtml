Agregar Columna

Para iniciar la asignación de columnas, seleccione un proveedor de la listar de proveedores como la de la siguiente imagen.

Al abrir la lista de proveedores, se verán de dos formas:

1. (Código proveedor) + Nombre proveedor Cuando se encuentra en la base de datos el nombre del proveedor se verá como se explica aquí.

2. Código proveedor Cuando no se encuentra en la base de datos el nombre del proveedor se verá como se explica aquí.

Una vez seleccione un proveedor, se habilitaran dos tags con las columnas disponibles y las columnas seleccionadas.

Columnas Disponibles

En este tag el usuario puede seleccionar una de las columnas para asignársela al proveedor; para saber más sobre columnas visite la ayuda de Columnas.

Si el usuario selecciona una de las columnas que son calculadas, aparecerá una ventana emergente que informara si el sistema necesita de otras columnas para poder ser agregada.

De clic en Aceptar para que el sistema agregue también las columnas mencionadas en la ventana emergente, o de clic en Cancelar para no hacer nada.

Seleccione las columnas que vea que son necesarias para el proveedor.

El usuario también puede filtrar las columnas, para encontrar más fácil la columna que se necesita.

Columnas Seleccionadas

El tag permite la gestión, ordenado, eliminación y exportación de las columnas.

El usuario puede usar las flechas identificadas con los siguientes iconos y, para ordenar y posicionar las columnas como desee que salgan en el documento de excel que podrá exportar dando clic en el botón de Exportar.

Después de dar clic en exportar, se generara el documento que se usara para la posterior proforma; el cual contendrá datos del proveedor, las columnas elegidas y los datos de la empresa.

Para quitar una columna del proveedor de clic sobre esta y de clic en el botón con el siguiente icono ; si la columna a eliminar es necesaria para el cálculo de otra, se desplegara una ventana emergente informando que no se puede eliminar la columna debido a que es necesaria; esto se puede ver en la siguiente imagen.

Si se quiere de igual forma eliminar esta columna, primero elimine la columna que está generando el conflicto y luego intente eliminar nuevamente la columna deseada.

El usuario también puede filtrar las columnas, para encontrar más fácil la columna que se necesita.
